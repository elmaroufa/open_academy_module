## Technologies :

---------------------------------------------------------------------------------------------------------------
 - Odoo 12
 - PostgreSQL 9.5 


## Installation et démarrage

L'environnement fonctionne sur Docker uniquement. Pour installer et démarrer, il suffit de faire:

```bash
docker-compose up
```

L'application est accessible sur **localhost:8069**
